# SnakeOS: An i386 bootloader and kernel that runs snake

![snake_preview](./preview.png)

# Building and running

Building the bootloader and kernel can be done using `make`. 
To run in a virtual machine, use `./qemu`

You will need `nasm`, `i686-elf-gcc`, `i686-elf-ld`, `i686-elf-objcopy`, and `qemu-system-i386`
